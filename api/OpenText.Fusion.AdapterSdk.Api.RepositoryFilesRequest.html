Remarks

File data retrieval is a part of the process of scanning repository. When the adapter receives that RetrieveFilesData command (RetrieveFilesDataAsync(RepositoryFilesRequest, IFileDataResultsHandler, CancellationToken) method), it should queue the binary contents and expensive metadata.
